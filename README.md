# 📊 BrightLearn Data Analytics — Research Assignment 1

> **Foundations of Data Analytics & Data Science**
> *A beginner's research journey into the world of data, AI, and machine learning.*

## 👋 About This Repository

This repository contains my handwritten research notes and answers for **Research Assignment 1** from the **BrightLearn Data Analytics & Data Science Bootcamp**.

The assignment covers foundational concepts across five key areas of data science and AI — researched, studied, and written in my own words as part of my learning journey into the data science field.

## 🎯 What This Assignment Covers

The assignment is divided into **5 sections** covering **37 questions** in total:

### 📁 Section A — Data Fundamentals
Exploring the building blocks of how data is stored, organised and managed.
- Databases, Data Warehouses, Data Lakes & Data Marts
- Structured vs Semi-Structured vs Unstructured Data
- Data Modeling, Data Mining & ETL processes
- DBMS platforms (MySQL, PostgreSQL, Oracle, MongoDB)
- Common storage formats: CSV, JSON, Parquet, Avro

### 🤖 Section B — Artificial Intelligence & Machine Learning
Understanding how machines learn and how AI works at a foundational level.
- Artificial Intelligence — Narrow AI vs General AI
- Machine Learning — Supervised, Unsupervised & Reinforcement Learning
- Deep Learning & Neural Networks
- Natural Language Processing (NLP)
- Large Language Models (LLMs), Generative AI & Computer Vision
- Training Datasets and the importance of data quality

### 🔧 Section C — APIs, Tools & Developer Concepts
Learning the tools and technologies used by data professionals every day.
- APIs, API Keys & REST APIs
- JSON & Prompt Engineering
- MAP functions in Python
- Cloud Computing — IaaS, PaaS, SaaS
- Version Control with Git
- Jupyter Notebooks & Python libraries (Pandas, NumPy, Matplotlib, Scikit-learn)

### ⚖️ Section D — Data Ethics, Governance & Careers
Understanding the responsible side of working with data.
- Data Privacy & GDPR
- Data Ethics & real-world examples of unethical data use
- Data Governance frameworks
- Bias in AI and its real-world consequences
- Career roles: Data Analyst vs Data Scientist vs Data Engineer vs AI Engineer
- Business Intelligence tools: Power BI & Tableau

### 💭 Section E — Personal Reflection
Personal reflections on the most interesting and challenging concepts, and how I see Data & AI shaping my future career.

## 💡 Key Concepts I Learned

Through completing this assignment, I developed a solid understanding of:

- ✅ The difference between databases, data warehouses, data lakes and data marts
- ✅ How machine learning powers modern AI systems
- ✅ The role of neural networks and deep learning
- ✅ How APIs work and why they're essential in software systems
- ✅ The importance of data ethics, privacy and governance
- ✅ Career pathways in data science — especially in healthcare and finance
- ✅ Core Python libraries used in data science work
- ✅ How predictive analytics uses past data to forecast future outcomes
